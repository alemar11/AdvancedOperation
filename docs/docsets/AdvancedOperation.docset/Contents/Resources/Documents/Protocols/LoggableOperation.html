LoggableOperation

public protocol LoggableOperation : Operation

Operations conforming to this protcol can log their most relevant status changes.

Warning

If using Swift 5.0 or lower run KVOCrashWorkaround.installFix() to solve some multithreading bugs in Swift’s KVO.
