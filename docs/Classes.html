  • AsynchronousOperation

    An abstract thread safe subclass of Operation to build asynchronous operations. Subclasses must override execute(completion:) to perform any work and call the completion handler to finish it.

  • DelayOperation

    An AsynchronousOperation that will simply wait for a given time interval, or until a specific Date. If the interval is negative, or the Date is in the past, then this operation immediately finishes.

    Note

    A DelayOperation cannot be cancelled once started running.
