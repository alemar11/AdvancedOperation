KVOOperationObserver

public final class KVOOperationObserver<T> where T : ObservableOperation

Observes Operation KVC compliant properties until the observed operation gets deinited.

  • observe(_:options:handler:)

    Observes changes for a KVC compilant Operation property via its keyPath definition.

    Note

    If you don’t need to know how a property has changed, omit the options parameter. Omitting the options parameter forgoes storing the new and old property values, which causes the oldValue and newValue properties to be nil.

    Declaration

    Swift

    public func observe<Value>(_ keyPath: KeyPath<T, Value>, options: NSKeyValueObservingOptions, handler: @escaping (T, NSKeyValueObservedChange<Value>) -> Void)

    Parameters

    keyPath

    The Operation KVC compliant property to observe.
    options

    Options to determine the values that are returned as part of the change dictionary. You can pass 0 if you require no change dictionary values.
    handler

    The block called when the property changes.
