Communicator iOS
===

### Project

Clone and install/update frameworks with:

`carthage bootstrap`

`carthage update --platform iOS`

To be sure that Carthage doesn't use a compiled version (downloaded from Github), use:

`carthage update --platform iOS --no-use-binaries`
